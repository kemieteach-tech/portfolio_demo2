# Yami 個人作品集網站

這是一個以日系水彩風格為主題的個人作品集網站，已改為使用 Tailwind CSS 與 daisyUI 進行樣式管理，保留原有圖片內容與版型層次。

## 專案內容
- `index.html`：首頁
- `projects/time-shop.html`：作品案例頁
- `assets/css/style.css`：自訂主題樣式與背景效果
- `assets/js/`：互動與動畫腳本
- `plan/`：網站規劃文件

## 開發方式
- 使用 Tailwind CDN 與 daisyUI 建構元件與主題樣式
- 透過自訂字型與背景圖案補足原本的插畫質感

## 本地預覽
可直接開啟 `index.html`，或使用本地伺服器（例如 `python -m http.server`）預覽。

## 部署方式
這個專案是靜態網站，可直接部署到 GitHub Pages、Netlify 或其他靜態托管服務。
