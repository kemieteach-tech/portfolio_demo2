# 日系水彩風插畫家個人作品集網站規劃書

本規劃書由資深網頁設計師兼前端工程師規劃，針對**日系水彩風、生活小物**定位的插畫家量身打造，兼顧極致的網頁效能（Core Web Vitals）、直覺的 UX 導向與 SEO 關鍵字佈局。

---

## 一、 網站架構與動線規劃 (Information Architecture)

### 1. 混合式架構策略
* **首頁採用「單頁式 (Single Page)」**：專注於品牌故事建立、風格一目了然的精選作品與高轉換率的行動呼籲 (CTA)。訪客能在 1 分鐘內快速掌握你的核心價值。
* **內頁採用「獨立作品頁 (Individual Project Pages)」**：點擊首頁作品後跳轉至獨立介紹頁。這是為了 **SEO 內容深度** 規劃，讓搜尋引擎爬蟲能透過文字理解你的插畫商業價值，並操作長尾關鍵字。

### 2. 首頁核心區塊規劃 (行銷與 SEO 導向)

| 區塊名稱 | 行銷目的 (UX) | SEO 佈局策略 (技術) |
| :--- | :--- | :--- |
| **Hero Section<br>首頁主視覺區** | 3 秒內決定生死。放置最具代表性的強烈插畫，搭配吸睛的個人定位標語。 | 佈局全站唯一的 `<h1>` 標籤。<br>圖片設定 `fetchpriority="high"` 提高 LCP 分數。 |
| **Featured Portfolio<br>精選作品區** | 精選 6-8 張不同類型（商業合作、個人創作、周邊商品）的縮圖，誘發點擊。 | 每個作品縮圖加上指向獨立作品頁的超連結，且圖片必須包含正確的 `alt` 說明。 |
| **About & Services<br>關於我與服務項目** | 建立信任感。列出你的插畫能為客戶解決什麼問題（如書籍封面、IP角色聯名）。 | 使用 `<h2>` 標籤，置入「插畫接案」、「商業合作」等高轉換意圖詞彙。 |
| **Testimonials & Brands<br>客戶見證與合作軌跡** | 社會認同。列出合作過的出版社、品牌 Logo 或客戶好評，證明是成熟的專業接案者。 | 運用結構化資料（Schema Markup）強化搜尋結果信任度。 |
| **CTA & Contact<br>行動呼籲與聯絡表單** | 臨門一腳。提供乾淨、好填寫的表單（詢價表單），減少複製 Email 的麻煩。 | 標明「預計回信時間」（如 24 小時內回覆），提高表單填寫率。 |

## 二、 關鍵字與 SEO 佈局策略

### 1. 核心關鍵字推薦（商業意圖導向）
1. **地區型關鍵字**：`台灣插畫家`、`台北插畫接案`（鎖定在地業主尋找外包）
2. **商業型關鍵字**：`商業插畫設計`、`商業插畫接案`（直接鎖定有預算的企業客戶）
3. **風格型關鍵字**：`日系水彩插畫`、`手繪風生活小物`、`透明感水彩設計`（吸引特定喜好的業主）
4. **應用型關鍵字**：`文創商品插畫設計`、`網頁插畫`、`書籍封面插畫`（鎖定特定場景的發案者）

### 2. 網頁標題與 Metadata 設定
* **首頁 Title Tag**：`Yami｜台灣商業插畫家・日系透明感水彩生活小物接案合作`
* **首頁 Description Tag**：`提供專業商業插畫設計、手繪風生活小物、網頁視覺與 IP 角色設計。專長日系透明感水彩風格，擁有豐富品牌合作經驗，歡迎各界商業合作與插畫接案洽詢。`
* **H1 標籤（全頁唯一）**：`<h1>Yami - 專注於日系水彩與生活小物的台灣商業插畫家</h1>` *(可用 CSS 進行視覺優化隱藏或優雅融合)*
* **H2 標籤示例**：`<h2>精選日系水彩作品案例</h2>`、`<h2>商業插畫服務項目</h2>`

### 3. 獨立作品頁 (Case Studies) 內容結構規劃
不要只放完稿圖，Google 喜歡有深度的脈絡故事：
1. **專案背景 (Overview)**：用 1-2 段文字說明與哪個品牌合作？客戶遇到了什麼痛點？
2. **設計概念與草圖 (Concept & Sketch)**：放上草圖或製程，文字描述「為什麼選擇這個色調與線條、如何呈現生活小物的溫度」。*（此處文字能被 Google 爬蟲抓取）*
3. **最終成果與成效 (Final Outcome & Impact)**：展示實體周邊、書籍封面實體照或網頁上線截圖，並提及客戶的正面反饋。

## 三、 網站全套文案庫

### 1. 首頁 Hero 區核心標語
* **大標題 (H1)**：用畫筆揉碎日光，把平凡的日子渲染成詩。
* **副標題**：你好，我是 Yami。專注於日系透明感水彩的插畫家，透過溫暖的手繪線條，紀錄生活小物的日常溫度。

### 2. 關於我 (About Me) 品牌故事
> **【H2 標題】在手寫的筆觸裡，找回生活溫柔的點滴**
>
> 每個人桌上那只冒著熱氣的馬克杯、昨晚剛讀完那一頁折角的書、窗邊隨著微風輕搖的綠植……這些看似微不足道的「生活小物」，都是我創作的靈感來源。
>
> 我是 Yami，一位著迷於日系水彩渲染與手繪質感的插畫家。我的創作風格強調水份與顏料交織出的「透明感」，保留了紙張微小的纖維起伏，以及色彩層次中留白的呼吸感。
>
> 對我而言，插畫不只是視覺的呈現，更是一種生活態度的延伸。在緊湊的現代節奏中，我希望能用帶有空氣感的日系手繪風，為商業品牌注入一股清新的日常感，也為觀者在繁忙的日常中，留下一抹溫暖而療癒的微光。

### 3. 作品案例 (Case Study) 通用填空範本
> **【專案名稱】：時光雜貨鋪 - 2026 手帳周邊商品插畫設計**
>
> * **專案背景 (Overview)**：
>   本次專案與【合作單位/個人品牌】合作，針對「生活感與日常陪伴」的核心概念，規劃一系列手帳周邊、貼紙與明信片插畫。希望能透過有溫度的手繪視覺，陪伴使用者記錄下生活中的每一天。
> * **設計概念 (Design Concept)**：
>   在風格上，我採用了經典的**日系透明感水彩**技法。挑選了復古咖啡機、文具、午後的甜點等生活小物作為主角。為了營造「歲月靜好」的氛圍，在色彩上刻意降低了飽和度，利用水彩自然的擴散與沉澱效果，營造出陽光灑落般的輕盈光影，讓每一件小物都彷彿擁有自己的故事。
> * **應用成效 (Impact)**：
>   完稿後除了印製實體周邊，亦延伸應用於社群視覺與官方網站 Banner。手繪特有的親和力與細緻度，成功提升了產品的感性價值，深受喜愛日系文創風格的受眾好評。

## 四、 前端效能與技術開發建議（直擊 Core Web Vitals）

由於插畫家網站存在「圖片量大」的天然劣勢，前端開發（建議使用 Bootstrap 5）時必須嚴格執行以下優化：

1. **圖片極致優化 (Image Optimization)**
   * **格式轉換**：捨棄傳統 PNG/JPG，全面轉為網頁新格式 **WebP** 或 **AVIF**，體積可減少 50% 以上。
   * **響應式圖片**：利用 HTML5 `<picture>` 標籤或 `srcset` 屬性，手機版載入細緻度適中的小圖，桌機版再載入大圖。
   * **延遲載入 (Lazy Loading)**：除首頁第一屏（Hero區）立刻載入外，其餘所有作品縮圖必須加上 `loading="lazy"` 屬性。
   * **圖示系統**：採用 Bootstrap Icons（CDN）作為全站圖示來源，讓按鈕、服務項目、聯絡資訊與作品導覽保持一致的視覺語言。

2. **字型載入優化 (Font Loading)**
   * 使用 Google Fonts 等中文字型時，必須在 CSS 中加上 `font-display: swap;`。確保在字型下載完成前，網頁能先用系統預設字型顯示文字，優化 CLS（累計版面配置轉移）指標。

3. **提升瀏覽器解析優先級 (Fetch Priority)**
   * 首頁最頂端的 Hero 區插畫，請加上 `fetchpriority="high"` 屬性，告訴瀏覽器優先下載該資源，能大幅縮短 LCP（最大內容繪製）時間。

4. **圖示與互動細節建議**
   * 導覽列、聯絡 CTA、作品卡片與服務區塊可使用 Bootstrap Icons 增加辨識度。
   * 建議圖示搭配 `bi` 類別與輕微的 hover 動畫，讓頁面更有精緻感但仍保持乾淨。
